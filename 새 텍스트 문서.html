정규분포 \(N(m_1, \sigma_1^2)\)를 따르는 확률변수 \(X\)와 정규분포 \(N(m_2, \sigma_2^2)\)를 따르는 확률변수 \(Y\)가 다음 조건을 만족시킨다. 모든 실수 \(x\)에 대하여 \(P(X \leq x) = P(X \geq 40 - x)\)이고 \(P(Y \leq x) = P(X \leq x + 10)\)이다. \(P(15 \leq X \leq 20) + P(15 \leq Y \leq 20)\)의 값을 오른쪽 표준정규분포표를 이용하여 구한 것이 0.4772일 때, \(m_1 + m_2\)의 값을 구하시오. (단, \(\sigma_1\)과 \(\sigma_2\)는 양수이다.) [4점]
